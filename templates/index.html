Odd Even Checker API


This is a simple API to check wether a following number is odd or even.

The result is displayed in the browser in an interactive manner in JSON format.


To check wether a number is even or odd, just go to the URL section of browser and after localhost address just write the following :- localhost_address/oddeven/number

for eg http://127.0.0.1:5000/oddeven/45

Then, the output which you will get will be =>

{ Number: 45, Result: "Odd" }


Make sure to add one number at a time. Don't add multiple number at same time.


After entering the number, press ENTER key to see the result in the browser./p>
